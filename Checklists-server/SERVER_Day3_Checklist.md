## TRAINING CHECKLIST

### 1. Independent Service Practice
- [ ] Trainee leads their own tables. Trainer handles larger parties.
- [ ] Trainee manages pacing, guest interaction, and order accuracy with minimal intervention
- [ ] "This is your section. Own it."

### 2. Menu Mastery & Beverage Deep Dive
- [ ] Detailed review of all menu items: ingredients, preparation, allergens, dietary flags
- [ ] Beverage deep dive: wine list, beer selection, signature cocktails, flavor profiles
- [ ] Food and drink pairings. Trainee should start making confident recommendations.
- [ ] "A knowledgeable server anticipates needs and guides the guest. That's the difference."

### 3. Advanced Service & Complex Scenarios
- [ ] Handling guest complaints, resolving service issues, maintaining composure under pressure
- [ ] Role-play: difficult guest, allergy miscommunication, wrong order, high-stress rush
- [ ] Special requests and dietary accommodations: know the difference between an allergy and a preference
- [ ] When to involve MOD. How to de-escalate without escalating.

### 4. Cross-Training Expo Session #1
- [ ] Expo station setup, expediting food orders, quality control, kitchen communication
- [ ] Trainee works expo 30-45 min. Understand how BOH flow impacts table timing.
- [ ] Once complete, return to assigned trainer for remainder of shift

### 5. Product Assessment Questionnaire (End of Shift: 10 min)
- [ ] Verbal quiz: trainer asks trainee to describe 5+ menu items, ingredients, and modifications
- [ ] Ad-lib format. Gauge retention and confidence. Note gaps for Day 4.

### 6. Closing Procedures
- [ ] Mid/closing checklist via Jolt app. Trainee performs with trainer observing.
- [ ] Tables fully reset, service areas stocked, restaurant prepped for next shift
- [ ] Closing reports: run server shift review on Toast, print, staple credit slips, submit to manager
- [ ] "Close the restaurant the way you'd want to find it when you open tomorrow."
