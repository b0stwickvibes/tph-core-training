## TEMPLATE RULES

**What's LOCKED (same on every checklist, every role, every day):**
- Header format (role, day number, day title, date/trainer/trainee fields)
- Checkbox format for every topic
- Arrow (→) key points under each checkbox — 1-2 lines max, reminders not instructions
- End-of-Shift Recap block (always last content section, always 10 min, always non-negotiable)

- Dual signature block with date

**What CHANGES per role and day:**
- Role name and day number in header
- Day title (descriptive — tells the trainer the theme of the day)
- Topic checkboxes and key points (pulled from Trainer Manual, day by day)
- CFU question in recap (specific to that day's content)
- Scenario prompt in recap (specific to that day's content)
- Tomorrow preview in recap (sets up next day's focus)

## FORMAT GUIDELINES

**Topic Checkboxes:**
- Each checkbox = one training topic from that day's Trainer Manual section
- Key points are reminders, not full instructions — the trainer already covered this in retraining
- 1-2 key points per topic. 3 max if it's a complex topic. Never more.
- Use quotes from training language where they reinforce the standard (e.g., "Your setup determines your success")
- Order follows the Trainer Manual's day sequence exactly — don't rearrange

**End-of-Shift Recap:**
- Always the last content block before the submission confirmations
- CFU (Check for Understanding) = a direct question the trainee should be able to answer based on today
- Scenario = a "what would you do" situation that tests application, not just recall
- Tomorrow preview = one sentence, sets expectations so the trainee shows up mentally prepared
- This block is ON the checklist so it can't be skipped — it's a line item, not a suggestion

**Submission Block:**
- Both checkboxes must be marked YES before trainer leaves the building
- Photo of this signed sheet uploads with the accountability form
- No submission = no trainer incentive for this shift

**Signatures:**
- Both trainer AND trainee sign
- Creates physical paper trail alongside digital form
- Trainee signature = confirmation they received the training listed above
