## TRAINING CHECKLIST

### 1. Independent Opening
- [ ] Trainee executes opening procedures via Jolt app with trainer observing
- [ ] Set up menus, iPads, host stand materials accurately and on time
- [ ] "A clean, organized host stand sets the tone for service."

### 2. Reservation Expertise (Resy)
- [ ] Create, adjust, and cancel bookings in Resy
- [ ] Block time slots, accommodate large parties, prevent overbooking
- [ ] Keep the system updated in real time throughout the shift

### 3. Answering Calls & Making Reservations
- [ ] Practice answering calls with professionalism and the standard greeting
- [ ] Gather: guest name, phone number, email, party size, special requests
- [ ] Confirm all details back to the guest before ending the call

### 4. Strategic Seating
- [ ] Balance reservation flow with walk-ins throughout the shift
- [ ] Table turnover pacing, section rotation, avoiding server overload
- [ ] "You control the flow of the dining room. Every seating decision matters."

### 5. To-Go Order Management
- [ ] Take to-go orders over the phone: collect name, number, repeat order back
- [ ] Process remote payments accurately. Confirm pickup time.
- [ ] "Mistake-free to-go orders build guest trust and repeat business."

### 6. Seating Guests
- [ ] Seat reservations at assigned tables, walk-ins at available tables
- [ ] Announce the guest to the server when seated, inform kitchen of active covers
- [ ] Greeting and demeanor should make every guest feel like a VIP

### 7. Team Coordination & Server Rotation
- [ ] Execute server rotation system for fair, balanced table distribution
- [ ] Coordinate seating with server sections and availability
- [ ] Communicate clearly: "I just sat you" verbal confirmation or system notification

### 8. Technology Utilization (iPad/Resy)
- [ ] Use Notes, Floor View, Waitlist, and Reservation tools inside Resy efficiently
- [ ] Keep the digital system updated in real time during service
- [ ] "The iPad is your control panel. Master it."

### 9. Phone Etiquette & Guest Communication
- [ ] Handle incoming calls during live service: reservations, to-go inquiries, general questions
- [ ] Manage call volume while greeting walk-in guests simultaneously
- [ ] Practice balancing phone professionalism with in-person warmth. Neither guest should feel rushed.

### 10. End-of-Shift Procedures
- [ ] Check in with MOD before clocking out. Ensure all host duties complete and systems updated.
- [ ] Resy updated for tomorrow: confirm upcoming reservations, clear completed waitlist entries
- [ ] Reflect with trainer on today's performance. Note areas needing more practice for Day 3.
