## PHASE 6: DUPLICATE CONTENT PROBLEM

**The issue you flagged:** Trainee checklist content is embedded inside the Trainer Manuals. Any change to trainee materials requires updating both documents. This is a maintenance headache that will only get worse as we scale.

**Options to explore (after core build is complete):**
1. **Decouple on Trainual** — Trainer Manual references the Trainee Manual by link instead of embedding it. Single source of truth for trainee content.
2. **Modular Trainual structure** — Break content into shared "steps" that both manuals pull from. Change once, updates everywhere.
3. **Accept it for now** — The restructure already reduces this problem because the Daily Floor Checklist (new Layer 2) replaces the embedded trainee checklist as the trainer's shift tool. The Trainer Manual becomes a retraining reference, not a daily-use document. Changes to trainee content still need to happen in both places, but the frequency of changes drops.

**Recommendation:** Option 3 in effect. The three-layer system reduces how often the Trainer Manual is touched day-to-day. Revisit after the restructure is live and stable.
